# Get It Done (GIT)

Professional service marketplace connecting skilled providers with customers globally.

## Features

- **Provider Dashboard** — See customer requests, accept jobs, track earnings
- **Customer Dashboard** — Post requests, see interested providers, book directly
- **Phone OTP + Email Login** — Two authentication options
- **Payment Integration** — Razorpay payment processing with commission tracking
- **Real-time Updates** — Requests and acceptances update instantly
- **Commission Model** — Fair pricing for providers, sustainable for platform

## Tech Stack

- **Frontend:** Next.js 14, React, TypeScript
- **Backend:** Next.js API Routes
- **Database:** Supabase (PostgreSQL)
- **Payment:** Razorpay
- **Auth:** Supabase Auth (Phone OTP + Email/Password)

## Getting Started

1. Create Supabase account (supabase.com)
2. Create Razorpay account (razorpay.com)
3. Setup environment variables in `.env.local`
4. Run `npm install && npm run dev`
